Clasificación de Dígitos MNIST – Visualización de la Inferencia

Visualización interactiva de una red neuronal

Esta aplicación muestra un perceptrón multicapa (MLP) compacto entrenado en MNIST. Dibuje un dígito y observe cómo las activaciones se propagan en tiempo real a través de todas las capas totalmente conectadas.

Cómo funciona:

  • Dibujar: Haga clic y arrastre en la cuadrícula 2D (arriba izquierda) para dibujar un dígito
  • Observar: Siga cómo su imagen atraviesa las capas de la red en el espacio 3D
  • Predicción: Revise la probabilidad de cada dígito (0–9) en el gráfico (arriba derecha)

Arquitectura de la red (exportación por defecto):

  • Capa de entrada: Rejilla de 28×28 píxeles (su dibujo)
  • Capa densa 1: 784 → 64 neuronas con ReLU
  • Capa densa 2: 64 → 32 neuronas con ReLU
  • Capa de salida: 32 → 10 logits → probabilidades Softmax

Controles 3D:

  • Rotar: Mantener presionado el botón izquierdo y arrastrar
  • Desplazar: Mantener presionado el botón derecho y arrastrar
  • Zoom: Usar la rueda del ratón

Código de colores:

  • Nodos: La coloración refleja la fuerza de activación (azules oscuros para valores bajos/negativos, corales intensos para activaciones positivas fuertes)
  • Conexiones: Colores cálidos representan contribuciones positivas fuertes; tonos fríos indican influencias negativas; líneas apagadas indican valores cercanos a cero.

Entrenar su propio modelo:

  • Ejecute python training/mlp_train.py para entrenar el MLP (incluye aceleración Apple Metal si está disponible).
  • El script genera exports/mlp_weights.json, que el visualizador carga al iniciar.
  • Modifique el número de neuronas, épocas o rutas de exportación mediante las opciones CLI documentadas en training/mlp_train.py.

Funciones en tiempo real:

  • Activaciones por capa: Las esferas muestran la activación de cada neurona usando colores.
  • Conexiones relevantes: Cada neurona destino resalta sus pesos de entrada más fuertes para mejorar la legibilidad.
  • Probabilidades en vivo: El gráfico de barras muestra los logits y las probabilidades Softmax en tiempo real.

La red se mantiene deliberadamente compacta para asegurar una visualización fluida. Puede entrenar arquitecturas más grandes, pero manténgalas ligeras para que la vista 3D siga siendo ágil.
